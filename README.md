# qstrike-tech-exam

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run Unit Test
```
npm run unit-test
```

### Alternatively, you can also run unit tests using
```
npx vitest
```
